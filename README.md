# Vrinda-Shop-Dashboard-Using-Excel
Data Cleaning, Data Processing, Data Analysis, Excel Dashboarding
Summary : Analyzed Vrinda Store’s sales data using Excel by cleaning, processing, and visualizing data to identify key business insights and trends.
Key Insights: Women made ~65% more purchases than men.
Top states: Maharashtra, Karnataka, Uttar Pradesh.
Highest contributing age group: 30–49 years. 
Best channels: Amazon, Flipkart, Myntra (~80% of sales).
